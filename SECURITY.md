# Security Policy

## Reporting a vulnerability

If you discover a security issue in this project (for example, in the deployment script or the site configuration), please report it privately.

- Preferred: use GitHub's [private vulnerability reporting](https://github.com/HannahVernon/agent-readme-md/security/advisories/new) for this repository.
- Alternatively, email **vuln@mvct.com** with a description, steps to reproduce, and any relevant details.

Please do not open a public issue for security reports.

We will acknowledge your report, investigate promptly, and keep you informed of progress toward a fix.

## Scope

This is a static website and specification.  The most likely areas of concern are the deployment script (`scripts/Deploy-Site.ps1`) and the web server configuration (`web.config`).  The specification prose itself is advisory content and is not a security boundary.
